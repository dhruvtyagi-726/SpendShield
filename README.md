# SpendShield AI

SpendShield AI is a personal finance fraud and expense intelligence dashboard built using Python, Pandas, Streamlit, and SQLite.

## Problem Statement

Many users do not regularly analyze their bank, card, or UPI transactions. Because of this, duplicate payments, recurring subscriptions, high-value transactions, and unusual expenses may go unnoticed.

SpendShield AI helps users understand spending patterns and detect basic suspicious transactions from transaction data.

## Features

- Transaction CSV analysis
- Category-wise spending summary
- Payment method analysis
- Duplicate payment detection
- High-value transaction detection
- Recurring subscription detection
- Saving suggestions
- SQLite database sync
- SQL query explorer

## Tech Stack

- Python
- Pandas
- Streamlit
- SQLite
- Rule-based transaction classification
